# CloudCrate

CloudCrate is a MERN-stack web application that simulates usage-based billing for object storage systems. The platform allows users to upload files, track storage consumption, and view billing information through a secure dashboard.

## Features

- User Registration and Login
- JWT-based Authentication
- Secure Password Hashing using bcrypt
- File Upload Management using Multer
- Storage Usage Tracking
- Usage-Based Billing Calculation
- Profile Management
- Password Update Functionality
- MongoDB Atlas Integration
- Frontend-Backend API Integration
